# Portfolio Watch|投資組合配置監控

監控「目標配置 vs 實際持倉」的偏離,並產生再平衡建議。支援多組合(各自獨立目標)、多帳戶、台股 + 美股 + 可擴充市場、多幣別自動換算。

- 前端:純靜態 PWA(GitHub Pages 託管,可加到手機主畫面)
- 報價:GitHub Actions 每日自動抓收盤價(台股 TWSE/TPEx、美股與匯率 Stooq)
- 資料:**持倉資料只存在你裝置的 localStorage**,repo 裡只有程式碼和公開股價,不含任何個人資料

## 部署步驟(約 5 分鐘)

1. 在 GitHub 建立 **公開** repo:`portfolio-watch`
2. 把整個專案內容 push 上去(根目錄就是 `index.html` 那一層)
3. 開啟 Pages:repo → Settings → Pages → Source 選 `Deploy from a branch`,Branch 選 `main` / `(root)`
4. 給 Actions 寫入權限:Settings → Actions → General → Workflow permissions → 勾選 **Read and write permissions**
5. 手動跑第一次報價:Actions → 「更新每日報價」→ **Run workflow**,成功後 `data/prices.json` 會有內容
6. 打開 `https://<你的帳號>.github.io/portfolio-watch/`,手機可「加入主畫面」

之後報價每個交易日自動更新兩次(台北 14:10 抓台股收盤、06:10 抓美股收盤與匯率)。

## 使用流程

1. **目標**頁:設定預計投入資金規模、每檔目標比例(建議把「現金」也設一個比例,合計 100%)
2. **持倉**頁:建立帳戶(含各幣別現金)→ 輸入持股部位(股數、平均成本)
3. **儀表板**:看配置尺(金色刻度 = 目標、金色區帶 = 容忍區間;紅 = 超配、綠 = 低配,台股慣例)
4. **再平衡**:「只買不賣」用現金補低配部位;「買賣皆可」完整回到目標
5. **總覽**:多組合並列(例:自己的 + 兒子的,各自獨立目標)

## 報價清單維護

編輯 `data/symbols.json`,把你持有的代號加進 `tw` / `us` 陣列後 push。沒在清單裡的標的可在持倉頁按「價」手動輸入價格(手動價與自動價並存時取較新者)。

想隱藏實際關注清單,可以把清單擴充成一籃子常見標的(例如台股權值前 100 + 常見 ETF),實際持股就藏在其中。

## 偏離判定規則

同時超過「絕對偏離」(預設 ±2 個百分點)與「相對偏離」(預設 ±25%)才警示,避免大部位太敏感、小部位永遠不警示。兩個門檻都可在設定頁調整。

## 常見問題

- **報價沒更新?** 到 Actions 看該次執行的 log;個別代號抓不到會沿用前次數值並在 log 註記。興櫃股票不在來源範圍,請用手動價。
- **複委託費率?** 設定頁可調,預設 0.5% 僅為概估,請改成你券商的實際費率。
- **換裝置?** 設定頁「匯出備份」→ 新裝置「匯入備份」。Google Drive 自動同步規劃於下一版。
